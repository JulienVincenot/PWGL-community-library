The Sound Description Interchange Format (SDIF) is an established standard for the well-defined and extensible interchange of a variety of sound descriptions including spectral, sinusoidal, time-domain, and higher-level models. SDIF consists of a basic data format framework and an extensible set of standard sound descriptions. The SDIF standard has been created in collaboration by Ircam, CNMAT, and IUA-UPF.

Mailing List

The SDIF mailing list allows discussions and provides information about new releases.
  • Use the sympa web interface, or:

  • To subscribe, send a mail to listserv@ircam.fr containing 
    subscribe sdif <first name> <last name>
  • To unsubscribe, send a mail to listserv@ircam.fr containing 
    signoff sdif [e-mail address if different from your reply-address]
  • Send messages to go to the list to sdif@ircam.fr.
